# Rcode Agent 能力地图

这个文档把 Rcode 后续能力拆成几个层级，方便持续对齐主流 coding agent 产品。

## 1. 项目上下文层

- 项目根目录识别
- 会话与项目绑定
- 项目规则说明
- 常用任务入口
- 最近任务记录

## 2. 模型接入层

- 默认通用免费模型入口
- 通用兼容协议接入
- 多 provider 切换
- 模型能力声明
- 流式输出能力
- 工具调用能力

## 3. Agent Runtime 层

- 多轮任务循环
- 工具调用队列
- 工具结果回填
- 失败恢复
- 中止与继续
- 任务状态展示

## 4. 工具系统层

- 文件读取
- 文件编辑
- 网页资料获取
- 本地任务执行
- 项目搜索
- Git 状态读取
- 测试结果读取
- 构建结果读取
- MCP 或插件扩展

## 5. 权限与安全层

- 请求批准
- 替我审批
- 完全访问
- 风险分级
- 项目边界判断
- 工具执行记录
- 用户可审计日志

## 6. 开发闭环层

- 任务规划
- 文件变更
- diff 审查
- patch 接受
- patch 撤销
- 测试反馈
- 提交建议

## 7. 产品体验层

- 设置页
- 模型管理
- 项目管理
- 会话管理
- 工具时间线
- 新手引导
- 桌面端体验

## 当前优先级

1. 先补设置页和模型 provider 管理。
2. 再补项目搜索与 Git 状态。
3. 再补 patch 接受、撤销和测试反馈。
4. 最后补 MCP、插件和多 agent 工作流。
