# Rcode 产品定位

Rcode 的产品定位是面向 Codex、Claude Code、Cursor Agent、GitHub Copilot Coding Agent 等市面主流 AI coding agent 的本地 Agent 客户端。

Rcode 不是单一模型供应商外壳，而是多模型、多协议、可控执行的本地 coding agent 产品。

当前仓库中的 Xiaomi MiMo 配置先作为默认通用免费模型入口，用于降低体验门槛。用户后续可以在设置中通过 OpenAI-compatible、OpenRouter、自定义 Base URL 等通用协议接入自己的模型服务。

## 方向

- 对齐 Codex 等主流 agent 的功能定位。
- 面向真实项目开发任务，而不是只做聊天助手。
- 保留本地控制、权限审批、工具调用过程和文件 diff 展示。
- 支持通用模型协议，让用户可以切换不同模型供应商。
